### 起動時に必要なもの
ディスプレイ(HDMI接続できるもの、もしくはタッチ・ミニディスプレイ)
intel MOVIDIUS (青いUSBスティック)
USB Web Camera (USB接続のカメラ)


### あると便利なもの
マウス
キーボード
USB拡張ハブ　(注意：カメラとMOVIDIUSスティックは本体に直接繋いでください)


### ミニディスプレイ 配線
B...R
|  /
B.R.................
....................


### そのた
電気マークが出るときは電力不足です。
本体電源が切れる場合は、電源コードやコンセントを確認してください。
電源コード：　5V / 2500mA 以上 (5V / 2.5A)
コンセント：　これに足る出力


### データ削除の呪文
せんたくしたデータをさくじょ


### 起動しない場合
起動時に必要なもの（上記）を確認して、アプリを起動してください。
本体を再起動してください。
電源、電力を確認してください。

それでも起動しない場合、詳細情報にヒントがあるかもしれません。
